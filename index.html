gdeploy

Overview

gdeploy is a tool to deploy and configure gluster clusters.

gdeploy uses ansible to deploy a cluster using a given configuration file. gdeploy can be used to set-up a backend for a GlusterFS volume, create a GlusterFS volume and mount it on one or more clients from an ansible installed machine.

Documentation

The gdeploy tool is documented in gdeploy(1) man page.
And gdeploy(5) documents in detail on writing a configuration file for gdeploy.
